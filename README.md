Halo! Terima kasih sudah menggunakan AI Image Upscaler. 
Aplikasi ini didesain untuk memperbesar resolusi foto kamu 
menjadi lebih tajam (HD) menggunakan teknologi AI (Real-ESRGAN).

-----------------------------------------------------------
📦 ISI FOLDER INI (JANGAN DIPISAH!)
-----------------------------------------------------------
Agar aplikasi berjalan lancar, pastikan 3 komponen ini 
selalu berada dalam SATU FOLDER yang sama:

1. [AI_Upscaler.exe]            -> Aplikasi utama (GUI).
2. [realesrgan-ncnn-vulkan.exe] -> Mesin AI (Otak aplikasi).
3. [models] (Folder)            -> Data kecerdasan buatan.

⚠️ PENTING: Jangan memindahkan file 'AI_Upscaler.exe' 
keluar dari folder ini. Jika dipisahkan, aplikasi tidak 
akan bisa menemukan mesin AI-nya.

-----------------------------------------------------------
🛠️ CARA PENGGUNAAN
-----------------------------------------------------------
1. Double klik pada 'AI_Upscaler.exe'.
2. Tarik (Drag & Drop) foto ke dalam jendela aplikasi.
3. Pilih Model AI:
   - x4plus: Untuk foto umum/pemandangan.
   - x4plus-anime: Khusus untuk gambar kartun/anime.
4. Klik tombol hijau "START UPSCALE".
5. Setelah 100%, gunakan tombol "Bandingkan" untuk melihat 
   hasil Before vs After secara interaktif.
